# Análise de URLs de Phishing

## Sobre o projeto

Este projeto tem como objetivo praticar SQL por meio da análise de um conjunto de dados de URLs de phishing.

Ao longo do desenvolvimento, serão criadas consultas SQL para explorar os dados e responder perguntas de negócio relacionadas às características que podem indicar se uma URL é maliciosa ou não.

### Tecnologias

- SQL
- SQLite
- DB Browser for SQLite
- Visual Studio Code
- Git e GitHub

**Status:** Em desenvolvimento.
